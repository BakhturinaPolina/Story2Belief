# notebooks/

Notebooks are for **exploration, demos, and figures only**. Anything that
needs to run as part of an experiment lives in `scripts/` or `src/`.

See [`../.cursor/rules/40-research-notebooks.mdc`](../.cursor/rules/40-research-notebooks.mdc)
for required structure.

## Index

- `01_narrative_pipeline_demo.ipynb` — load a sample text, run Layer 1,
  inspect the resulting `NarrativeFeatures`.
- `02_reader_cohort_demo.ipynb` — sample a cohort, render a few
  agent personas, run the reading loop on one text.
- `03_bridge_demo.ipynb` — run the pre/post dilemma test on a single
  reader and visualize the deltas.
- `04_end_to_end_prototype.ipynb` — committee-facing demo notebook
  combining all three.

Each `.ipynb` is currently a placeholder; populate during the relevant
prototype week (see [`../docs/02-prototype-roadmap-3mo.md`](../docs/02-prototype-roadmap-3mo.md)).
