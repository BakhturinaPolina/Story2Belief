# Local reference PDFs (not in git)

These files support dissertation planning and instrument design. They
are **gitignored** and must be obtained locally; they are not published
on GitHub.

| File | Description |
|------|-------------|
| [`kuijpers-2014-swas.pdf`](kuijpers-2014-swas.pdf) | Kuijpers et al. (2014) — SWAS development paper (*Scientific Study of Literature* 4(1)). Canonical citation for [`configs/instruments/swas.yaml`](../configs/instruments/swas.yaml). OSF scoring script: <https://osf.io/mhvtb/>. |
| [`liwc-22-manual.pdf`](liwc-22-manual.pdf) | Boyd et al. (2022) — LIWC-22 development manual. Evaluated and **excluded from the prototype**; see [`12-tools-and-resources.md`](../12-tools-and-resources.md). |
| [`core-narrative-transportation-persuasion-phd.pdf`](core-narrative-transportation-persuasion-phd.pdf) | Internal literature synthesis on narrative transportation and persuasion (variable selection for Layer 1 and Layer 2). |

If a file is missing, check your backup or re-download from the original
source before citing paths in docs or configs.
